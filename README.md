# KalaVerse - Django E-Commerce Website

## Overview
KalaVerse is a Django-based e-commerce website designed for artists and art enthusiasts. It allows users to browse products, add them to an art bag (cart), place orders, and manage their purchases.

## Features
- User Registration and Login
- Product Listing
- Product Details
- Add to Cart
- Remove from Cart
- Checkout
- Order History
- Admin Panel
- Product Images

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite
- Git & GitHub

## How to Run

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the server

```bash
python manage.py migrate
python manage.py runserver
```

Open:
http://127.0.0.1:8000/
